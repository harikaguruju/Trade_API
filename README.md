# 📊 AI Market Analysis Dashboard

An AI-powered web application that analyzes market sectors and generates structured business insights using real-time data and generative AI.

---

## 🚀 Features

- 🔍 Sector-based market analysis (e.g., healthcare, technology)
- 🤖 AI-generated insights (trends, opportunities, risks)
- 📈 Interactive charts (growth, risk, demand, trend analysis)
- 🏢 Company insights with logos
- 📊 Market size and growth rate
- 📄 PDF report download
- 🔐 Simple login system
- ⚡ Real-time API-based analysis

---

## 🛠️ Tech Stack

- **Backend:** FastAPI (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Charts:** Chart.js
- **AI:** Gemini API
- **Deployment:** Render
